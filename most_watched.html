  • This open-source YouTube analytics dashboard was developed for EOTC Debre Berhan (ደብረ ብርሃን) community church to provide clear, data-driven insights into channel performance and audience engagement.
  • The project integrates the YouTube Data API to collect channel and video-level metrics through a Python-based data pipeline.
  • SQLite is used for structured storage, transformation, and ETL processes, ensuring reliable and efficient data management.
  • The interactive dashboard is built in R, presenting growth trends, engagement rates, and content performance in an accessible and actionable format.
  • Hosted on GitHub, the repository includes automated weekly data refreshes powered by GitHub Actions, enabling the dashboard to remain up-to-date without manual intervention.

  • Check out the project on Github: https://github.com/yihun/eotc-youtube-channels
